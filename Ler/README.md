# Exercícios de Lógica de Programação – Orientação a Objetos  
Repositório com diversos exercícios voltados para programação Orientação a Objetos (POO): classes, objetos, herança, polimorfismo, interfaces, entre outros.

## Sobre  
Este projeto reúne múltiplos desafios de lógica de programação com foco em **Orientação a Objetos**: modelagem de entidades, comportamento, estratégias, composição e herança. O objetivo é ajudar a consolidar os conceitos de POO em Java de forma prática e com exemplos simples.

## Tecnologias utilizadas  
- Java 
- Ambiente de desenvolvimento Java padrão (JDK)  
- Sem framework específico — foco na linguagem e nos conceitos de OO  

## Exemplos de arquivos presentes:
- `Aluno.java` — representa uma entidade aluno com atributos, métodos, etc.  
- `Animal.java` — hierarquia de classes animais, uso de herança/polimorfismo.  
- `StrategyGolpe.java`, `StrategyPagamento.java` — demonstração de padrão Strategy.  
- `Livro.java`, `Relatorio.java`, `Computador.java` — outros exemplos de modelagem orientada a objetos.
